- 공부 컨셉(1) (헨젤과 그레텔 스타일)
  1. 과제를 해결하면서, 다른 사람의 코드와 도움을 받는다.
  2. 대신 무조건 학습내용을 내 것으로 만들기위한 **Application** 또는 **report**을 만든다.
  
- 공부 컨셉(2) (콜롬버스 스타일)
  1. 과제를 해결한 이후에, 다른 사람의 코드와 도움을 받는다.
  2. 진도가 느리더라도 프로그래머로써 새로운 것을 해결해나가는 방법/자세를 배운다고 생각한다.
